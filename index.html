About the project

Built inside
Rec Room

This project archives the internal API data from two private server builds — one from 2021 and one from 2023 — of Rec Room VR. The 2021 data captures the official game APIs from that era, while the 2023 files come from the Vanadium / Reloxa private server project.

All data is preserved as-is, giving a clear window into how Rec Room's internal systems worked — from how loading screen tips were targeted by platform, to how game doors were configured with search queries.

The Pterodactyl build itself was a full-scale VR creature built entirely in Rec Room using the maker pen and shape tools — no pre-built assets.

History

Project timeline

From the first build to the API archive.

2021
Original Build & APIs

The Pterodactyl was built in Rec Room VR using only the maker pen and shape tools. The 2021 API data — 194 loading screens, 1,231 charades words, game configs — was captured from the live game at this time.

2022
Private Server Development

Work began on the Vanadium / Reloxa private server project, reverse-engineering Rec Room's API layer and building a custom backend to host sessions independently.

2023
2023 API Set Released

The updated 2023 API files were created for the Vanadium build — featuring 398 new game config keys, custom loading screen tips, the community board, and a redesigned CDN config.

Now
Open Archive

All files are published openly on GitHub. This site serves as the homepage and documentation for the full data archive.
